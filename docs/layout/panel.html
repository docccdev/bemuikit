Panel

Прямоугольный блок с разными темами и внутренними отступами.

UIKit использует панели для того чтобы наметить определенные разделы вашего содержания, которые могут быть стилизованы по-разному. Как правило, панели расположены в столбцах сетки из компонента Grid.

Использование

Блок состоит из самой панели .ui-panel и дочерним элементом .ui-panel__body . По умолчанию панель не имеет цвета и отступов.
Class
Description
.ui-panel
Добавьте этот класс для элемента, чтобы определить компонент панели.
.ui-panel_body
Добавьте этот класс дочернему элементу для создания тела панели.

Размеры

Модификатор размера панели определяет внутренние отступы. По умолчанию, панель не имеет отступов.
Class
Description
.ui-panel_size_ size
Внутренний отступ.

</div>

Темы

Определяется фоновым цветом. По умолчанию, панель не имеет фона.
Class
Description
.ui-panel_theme_transparent
Цвет фона прозрачный.
.ui-panel_theme_gray
Цвет фона серый.
.ui-panel_theme_white
Цвет фона белый.
.ui-panel_theme_blue
Цвет фона синий.
